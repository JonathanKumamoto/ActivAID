Deleting a user

Use the Login Manager to delete a user from the application.

  1. Click the delete user button in the upper left corner or right-click the User ID in the user list and select "Delete User".
  1. When "Delete User" is selected, a message box appears to confirm deletion.  Select "Yes" to delete user, select "No" to keep user in the application.
  2. The user will be permanently deleted once the "Yes" button is selected.
