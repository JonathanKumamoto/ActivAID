Window Menu

The Window menu provides management for multiple open documents, each of which exists within its own tab in the ActivATE IDE.  All open documents are listed at the bottom of the menu.  A checkmark is placed beside the document that currently has input focus.  Click on a document in this list to bring it to the forefront of the application.  The other menu items are discussed in detail below.

 

 New Horizontal Tab Group  New Vertical Tab Group  Close All Documents  Reset Window Layout

 

New Horizontal Tab Group

Creates a new horizontal tab group so that two documents can be viewed simultaneously.  This option is only available if more than one document is open.  A new horizontal tab group can also be created by right-clicking on an open tab and selecting "New Horizontal Tab Group" from the context menu.  To move a tab back to its previous location, right-click on the tab and select "Move to Previous Tab Group" or click on the tab and drag it to another tab group.

Example:  Two test programs are visible after moving "MyTest" to a new horizontal tab group.

New Vertical Tab Group

Creates a new vertical tab group so that two documents can be viewed simultaneously.  This option is only available if more than one document is open.  A new vertical tab group can also be created by right-clicking on an open tab and selecting "New Vertical Tab Group" from the context menu.  To move a tab back to its previous location, right-click on the tab and select "Move to Previous Tab Group" or click on the tab and drag it to another tab group.

Example:  Two test programs are visible after moving "MyTest" to a new vertical tab group.

Close All Documents

Closes all open documents.  If a document has unsaved changes, the user will be prompted to save the file before it is closed.

Reset Window Layout

Resets all windows within the application to their default size and location.
