Viewing the Current Drivers

To view the current set of drivers that are available within the ActivATE Test Platform, click on the "Configuration" pane in the Outlook-style toolbar.  The tree view shows three groups: Devices, Controls and Tools.  Devices are ActivATE™-specific instrument drivers.  Controls are the GUIs, or Graphical User Interfaces, that communicate with the devices.  Tools can be non-ActivATE™-specific instrument drivers, or they can be non-instrument "tools", such as a math or Excel module.  To view the drivers in each category, simply expand the tree view to see a complete listing.  Expanding the individual drivers in the tree view will allow you to see all the public APIs (methods and properties) that are available for each driver.  Hovering the mouse over the APIs in the tree view will display a tool tip that shows the data type of properties and the return type and parameter types of methods.

The tree view shows the drivers that are available in the application.
Expand a node to view a list of all the public APIs that are available in each driver class.  Hover over an API to see the return type and method parameters.
