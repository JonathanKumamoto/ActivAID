Test Program Templates

To create a new test program from a template:

  1. Select 'New Test Program' from either the File menu or the toolbar.
  2. Enter a name, description, etc. as usual for your new test program in the New Test Program dialog box.
  3. Check the "Use Template" checkbox.  The drop-down box is enabled and populated with all templates that are located in the Templates directory.

  1. Select the template file to use from the drop-down box.
  2. Click 'OK' when you are finished to create the test program.

Tip: The template selection is persisted and will appear as the default the next time a new test program is created.
