Test Results

The results of each test in a test program are automatically collected every time a program is run.  You may choose to view these results from the ActivATE Integrated Development Environment (IDE) and/or save the results to a file for later use.  The test results settings are user-configurable and are defined from the Properties Page in the IDE and from the test script.  

 

WARNING:  If an invalid printer is set as the computer's default printer, the test results processing at the end of a test program may take several minutes and will delay the system's ability to start a new test.  If this problem occurs, verify that the default printer is correctly configured.
