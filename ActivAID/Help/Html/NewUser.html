Adding a new user

Use the Login Manager to add a new user account for the application. 

  1. Click the new user button in the upper left corner or right-click on an empty line in the user list and select Add New User.
  1. The fields on the left side of the form will now be enabled.  Enter the User ID for the new account.  The User ID must be unique.
  2. Enter a description (optional).
  3. Enter the password in the Password field.  Check the Show Password as Text During Edit to show the real password characters instead of asterisks (*).
  4. Re-enter the same password in the Confirm Password field.
  5. Select the group the user is to be part of.
  6. Click the Create User button to save the new user.  If successful, the new user account will appear in the user list.
