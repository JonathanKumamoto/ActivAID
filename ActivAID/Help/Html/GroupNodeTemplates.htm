Group Node Templates

To create a new group node from a template:

  1. Right click on the node to add a group node to in the Test Sequence View and select 'New Group'.
  2. Enter a name, description, etc. as usual for your new group node in the New Group dialog box.
  3. Check the "Use Template" checkbox.  The drop-down box is enabled and populated with all templates that are located in the Templates directory.

  1. Select the template file to use from the drop-down box.
  2. Click 'OK' when you are finished to create the group.

Tip: The template selection is persisted and will appear as the default the next time a new group node is created.
