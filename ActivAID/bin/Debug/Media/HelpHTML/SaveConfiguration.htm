Saving a Configuration

You may have the need to switch back and forth between different driver configurations on the same system.  In this case, it is best to save each configuration as a named configuration that can easily be switched in and out of the environment. 

Save the Current Configuration | Load a Saved Configuration

Save the Current Configuration

  1. Right-click in the tree view on the Configuration pane and select "Manage Configurations" from the context menu.


     
  2. The Manage Configuration dialog appears.  Enter a name to save the current configuration as in the text box under Save Current Config:


     
  3. Click the Save button to save the configuration.

 

Load a Saved Configuration

  1. Follow step 1 above to show the Manage Configurations dialog.
     
  2. The saved configurations that are available will be listed in the drop down box under Load Saved Configuration.


     
  3. Select the configuration to load and click the Load button. 

    Note: If you would like to save the current configuration, please do so before loading a new configuration.
     
  4. Once the saved configuration has been successfully loaded, the Reinitialize button on the toolbar will be highlighted in yellow as shown below.  The loaded configuration will NOT take effect until the environment has been reinitialized or until the application is restarted.  To reinitialize, make sure all test programs and driver UI's are closed and then click the Reinitialize button.
