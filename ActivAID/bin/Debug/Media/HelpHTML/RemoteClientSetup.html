Remote Client Setup Menu

Accessing the Remote Client Setup menu provides users with a new remoting feature that allows for two systems to communicate with each other to use the same groups of slots and devices for testing.  A group of systems exist together in a client-server relationship.  For each group, there is one server that each client connects to.  It is the only system that has the ability to assign and release slots to each client.  By default, a user's computer acts as a server.  To access the remoting feature settings, a user must have admin priveleges and must access the remote client setup menu at the top main toolbar of ActivATE.  From this menu, a user can change their role between client and server.
