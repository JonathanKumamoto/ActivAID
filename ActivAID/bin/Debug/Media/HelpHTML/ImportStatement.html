Using the Import statement

The Import statement is used to include additional VBScript files for use by the test program during runtime.  These files may be located in the current test directory, in another location on the local computer, or on a network drive.  Multiple Import statements may be used, and the imported files may also contain additional Import statements.  Generally, the imported files contain methods and variables that are commonly used among multiple test programs.  This saves the user from having to recreate this code for every test program.

When a test program is run, the runtime engine searches for Import statements and creates an object for each imported file.  These objects can then be used in the main test script to access any public methods and variables in the imported files.  

Syntax

Import "<File>", "<ScriptName>"

File: The full path of the file to be imported.  Note: If the file is located in the current test program directory, 
the path may be omitted and just the file name needs to be referenced. 
ScriptName:  The name of the object to be used to reference the imported file from the main test script.

Although the only requirement is that any Import statements must be declared within the first 50 lines of code, in general it is best to declare them at the top of the file immediately following the Option Explicit statement.  (If used, the Option Explicit statement must always be the first line of executable code.)

Example:  Import a file named MathStuff.tpf from the current test directory.

  Option Explicit       ' MUST be declared on the first line
   
  Import "MathStuff.tpf", "math"     
     .
     .
     .

  ' Call a function declared inside MathStuff.tpf
  Dim x, y, z
  x = 5 : y = 10
  z = math.Average(x, y)
 

The object name "math" is used here to reference the Average function defined inside MathStuff.tpf.  In this example, MathStuff.tpf consists of the following code:

' Returns the average of x and y
Function Average(x, y)
	Average = (x + y)/2
End Function
' Returns the sum of x and y
Function Sum(x, y)
	Sum = x + y
End Function
' Returns the difference between x and y
Function Difference(x, y)
	Difference = x - y
End Function
