Driver Configuration

The ActivATE™ Test Platform's default configuration consists of several instrument drivers, such as a DMM, switch card and generic GPIB/VXI driver.  The driver set can easily be configured to meet the requirements of the user's test application.  Additional ActivATE™ drivers are available in the 'Drivers' directory (usually located at "C:\Program Files\Racal Instruments\ActivATE .Net 3.x\Drivers").  The user may also create their own drivers, either using the ActivATE™ driver wizard for Visual Studio .NET (preferred) or creating a .DLL or .EXE file from scratch.  The ActivATE™ test sequencer can directly consume .NET and COM assemblies, and Win32 assemblies may be used by writing a .NET wrapper assembly around the Win32 target.

The Configuration pane on the Outlook-style toolbar provides the functionality necessary to add, remove and modify drivers.
