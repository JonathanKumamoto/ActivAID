Editing a user

Use the Login Manager to Edit a user account for the application.

  1. Click the edit user button in the upper left hand corner or right-click on the User ID to edit in the user list and select "Edit User".
  1. Once "Edit User" is selected the fields on the left side that can be edited will be enabled.  Fields that can be edited are "Description", "Password", and "Member Of".  The User ID cannot be edited.
  2. Click the "Update User" button when editing is complete to save updated user information.  Fields will become disabled again after the changes have been successfully been changed.
