Options Dialog

The options dialog can be accessed from the Options menu item in the Tools menu and is used to configure various settings within the ActivATE™ application.  The options dialog will be expanded as new features are added.

Startup

Customizes the Web Site/HTML File/RSS Feed that is seen when the ActivATE application is launched, based on the URL given.  If the "Show Start Page on startup" checkbox is left unchecked, the Start Page will become blank the next time ActivATE is launched. 

For more customizations options, see: Start Page Customization

Application Path Settings

The Application Path Setting defaults can be changed from the Options→Application Path Settings option on the Options Dialog Box.  These path settings may not take effect until ActivATE is restarted.

Protect Files

These settings check to see if the test programs have been changed before they have been executed.  If the test programs are protected, they will not run if there are pending unsaved changes.  See the Protect Files help topic.
