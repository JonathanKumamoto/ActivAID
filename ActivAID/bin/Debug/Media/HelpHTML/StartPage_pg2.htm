Start Page Customization

The start page is shown by default when the ActivATE™ application is launched.  It has links to recently accessed documents and to resources such as documentation and email support.  It also features a customizable title bar and customizable logo ICON.

Users may customize the Title, Logo, and the URL.  The URL is simply an HTML file (such as a ReadMe.HTML), an actual internet URL, or even an RSS feed.  To change the default startup URL, go to the ActivATE menu item Tools->Options->Startup.

The Recent Documents shows the last 5 documents loaded into the ActivATE editor.  Right-Mouse-Click on any document to remove this entry from the MRU (Most Recently Used) document list or to open the actual file location where this file is located.

To change the title bar on the ActivATE start page,  right-mouse-click anywhere on the Title Bar and select "Change Start Page Title".

A user-definable title can be typed into the edit box as shown.

The new title is immediately refreshed.

The Start Page ICON can also be changed in a similar manner.

After being changed to a user-defined ICON, the menu item will reflect the new ICON path.  Most any picture file will work (BMP, PNG, JPG, ICO, etc).

The Start Page configuration file is an XML file stored in the ActivATE Config dir under the name of StartPage.config. This file can also be edited by hand or manually copied to this directory.
