Properties Page

By default, the Properties Page window is docked in the bottom left corner of the IDE.  The Properties Page window is visible by default, but it can be toggled from the View menu (View -> Properties Page).  It is used to get and set various properties of the test program, group nodes and test nodes.  The properties of a particular node are displayed when that node is clicked in the Test Sequence View.  

The properties are grouped by category.  Some properties are read-only and appear grayed-out.  These properties cannot be modified and only provide information.  Other properties, depending on the type of property, can be modified by selecting True or False, selecting a pre-defined value from a drop-down list, or typing in a string value.  Click on a property to view a more detailed description of it in the help box at the bottom of the Properties Page.
