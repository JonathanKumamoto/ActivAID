Deleting a Driver

To delete a driver from the current configuration, right-click on the driver to be deleted in the tree view on the Configuration pane and select "Delete" from the context menu. 

If a driver under the Devices category is deleted and this driver had an associated UI object under the Controls category, both the device and the control will be deleted.  If a driver under the Controls category is deleted, only the UI itself will be removed.  The device driver will still be present under the Devices category. 

Important:  Committing Changes

After a driver has been deleted, the Reinitialize button on the toolbar will be highlighted in yellow as shown below.  The deleted driver will still be available until the environment has been reinitialized or until the application is restarted.  To reinitialize, make sure all test programs and driver UI's are closed and then click the Reinitialize button.

If at any time you wish to cancel your changes, click the Rollback button at the bottom of the properties page.  This will undo all changes that have been made since the last reinitialization.  The Rollback button is only enabled if there are uncommitted changes (until the system is reinitialized or until the application is shut down).
